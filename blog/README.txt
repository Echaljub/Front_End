A blog page designed with three spots for ads. The articles links lead to
single.html and the logo back to the main page. The page was designed using 
bootstrap5. The forms and buttons have no functionality, but the social network
icons leads to the the respective main page.
The page was done following the instructions and using the
resources provided at the course "Diseño Web Profesional El Curso Completo,
Práctico y desde 0" by falconmasters.
